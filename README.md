## This is the Snakemake (7.18.1) workflow for studying RNA editing at the single-cell level

## Requirements
- Conda
- Docker

## The workflow structure
The workflow is consist of several sub-workflows (under the directory workflows/*).

## Usage
The final outputs are generated by calling the rule `all` in Snakefile.

The recommended way of executing the workflow is using the `run` command from [here](https://github.com/jerry955071/mytools/blob/main/run):

```{bash}
# run the workflow (replace <n_cpu> by the number of CPUs for Snakemake)
run Snakefile <n_cpu> all
```

This command expects a conda environment named `snakemake` with Snakemake (7.18.1) installed on your PC. Which can be create using:

```{bash}
conda create -c bioconda -c conda-forge -n snakemake snakemake=7.18.1
```
